## Online English Reading Grading Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/31d65c59c513468aaa81ae67b9c0c6b0/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Announcements
5. Books
6. Personal Center
7. My Bookshelf
8. Chapter Favorites
9. Browsing History


**Administrator Functions:**
1. Login
2. Personal Center (modify password, personal information)
3. Administrator Management
4. Basic Data Management (announcement type management, book type management)
5. Announcement Management
6. Book Management
7. Book Comment Management
8. Chapter Management
9. Chapter Comment Management
10. Chapter Favorites Management
11. User Management
12. Carousel Management
